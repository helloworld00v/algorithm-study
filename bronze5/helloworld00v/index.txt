# [1000] 두 정수 a와 b를 입력받은 다음, A+B를 출력하는 프로그램 작성
# 입력 : 첫째 줄에 A와 B가 주어짐 ( 0<A, B<10 )
# 출력 : 첫째 줄에 A+B를 출력한다
a, b = input().split()    # 해당 코드를 통해 a, b 변수에 입력받은 값을 저장
print(int(a)+int(b))      # split함수를 사용하게되면 문자열을 공백기준으로 잘라서 저장하므로
                         # 변수 a와 b에는 각각의 입력값이 저장
                         # 이제 변수 a와 b를 더해주어야하는데, 이때 각 변수는 문자형 자료형으로 저장되어있으므로
                         # 연산처리를 해주기 위해서는 정수형 자료형으로 변환해야!!
                         # int함수를 이용해 각 변수를 정수화시켜준뒤 계산
                         # 각 변수가 문자형 자료형으로 되어있는 이유 (split함수를 통해 이미 우리가
                         # 입력받은 값의 (문자열)값의 공백을 기준으로 띄어서 인식하였기 때문

a, b = map(int, input().split())        # map(f, iterable)은 함수(f)와 반복 가능한(iterable) 자료형을 입력으로 받는다.
                                        # map은 입력받은 자료형의 각 요소가 함수f에 의해 수행된 결과를 묶어서 리턴하는 함수
print(a+b)            # split() : 괄호 안에 아무런 값도 넣어 주지 않으면 공백(스페이스, 탭, 엔터 등)을 기준으로 문자열을 나누어준다.
                     #          : 괄호 안에 특정한 값이 있을 경우에는 괄호 안의 값을 구분자로 해서 문자열을 나누어준다.

# [1001] 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.
# 첫째 줄에 A와 B가 주어진다 ( 0<A, B<10 )
# 첫째 줄에 A-B를 출력한다.
a, b = input().split()
print(int(a) - int(b))

a,b = map(int, input().split())
print(a-b)
